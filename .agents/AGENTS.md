# Project Rules for Real Estate Workspace

*   **No Auto-Pushing to Git**: Do not run `git push` automatically. Only commit locally, and wait for the user's explicit request to upload/deploy to GitHub.
*   **Direct Execution without Blocking Plans**: Do not block the user with implementation plans or request feedback via artifacts (`RequestFeedback: true`) for coding tweaks. Perform the coding tasks directly and present the final results when complete.
